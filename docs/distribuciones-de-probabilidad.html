13 Distribuciones de probabilidad

13.1 Concepto teórico

Una distribución de probabilidad describe el comportamiento probabilístico de una variable aleatoria. En ingeniería, permite modelar fenómenos aleatorios y estimar la probabilidad de distintos resultados.

13.2 Distribuciones discretas

13.2.1 Distribución Bernoulli

Describe experimentos con dos resultados posibles: éxito o fracaso.

\[ P(X=x)=p^x(1-p)^{1-x}, \qquad x\in\{0,1\} \]

13.2.2 Distribución Binomial

Modela el número de éxitos en \(n\) ensayos independientes con probabilidad \(p\) de éxito.

\[ P(X=k)=\binom{n}{k}p^k(1-p)^{n-k} \]

Valor esperado y varianza:

\[ E(X)=np,\qquad Var(X)=np(1-p) \]

13.3 Distribuciones continuas

13.3.1 Distribución Normal

Es una de las más importantes en estadística e ingeniería. Su función de densidad es:

\[ f(x)=\frac{1}{\sigma\sqrt{2\pi}}e^{-\frac{(x-\mu)^2}{2\sigma^2}} \]

donde: - \(\mu\): media - \(\sigma\): desviación estándar

13.4 Ejemplo aplicado

13.4.1 Distribución binomial

Supóngase que la probabilidad de que un panel solar presente una microfalla es \(p=0.10\). Se inspeccionan \(n=8\) paneles y se desea conocer la probabilidad de encontrar exactamente 2 con microfalla.

13.5 Implementación en R

dbinom(2, size = 8, prob = 0.10)
## [1] 0.1488035

13.5.1 Probabilidad acumulada binomial

pbinom(2, size = 8, prob = 0.10)
## [1] 0.9619082

13.5.2 Esperanza y varianza binomial

n <- 8
p <- 0.10
c(Media = n * p, Varianza = n * p * (1 - p))
##    Media Varianza 
##     0.80     0.72

13.6 Gráfica binomial

k <- 0:8
prob_bin <- dbinom(k, size = 8, prob = 0.10)

barplot(prob_bin,
        names.arg = k,
        col = "lightblue",
        main = "Distribución binomial",
        xlab = "Número de paneles con microfalla",
        ylab = "Probabilidad")

13.7 Ejemplo con distribución normal

Se supone que el voltaje de operación de un sistema sigue una distribución normal con media 220 V y desviación estándar 3 V.

13.7.1 Probabilidad de que el voltaje sea menor que 218 V

pnorm(218, mean = 220, sd = 3)
## [1] 0.2524925

13.7.2 Probabilidad de que el voltaje esté entre 218 V y 223 V

pnorm(223, mean = 220, sd = 3) - pnorm(218, mean = 220, sd = 3)
## [1] 0.5888522

13.8 Gráfica normal

x <- seq(210, 230, by = 0.1)
y <- dnorm(x, mean = 220, sd = 3)

plot(x, y, type = "l", lwd = 2, col = "blue",
     main = "Distribución normal del voltaje",
     xlab = "Voltaje (V)", ylab = "Densidad")
abline(v = 220, col = "red", lty = 2)

13.9 Simulación

13.9.1 Simulación binomial

set.seed(111)
sim_bin <- rbinom(1000, size = 8, prob = 0.10)
mean(sim_bin)
## [1] 0.797
var(sim_bin)
## [1] 0.6704615

13.9.2 Simulación normal

set.seed(222)
sim_norm <- rnorm(1000, mean = 220, sd = 3)
mean(sim_norm)
## [1] 219.9532
sd(sim_norm)
## [1] 2.97975
hist(sim_norm, breaks = 20, col = "lightgray",
     main = "Simulación normal del voltaje",
     xlab = "Voltaje (V)")

13.11 Caso real (ingeniería)

En ingeniería industrial, la distribución binomial puede modelar defectos por lote y la normal, tiempos de producción o dimensiones de piezas.

En ingeniería civil, la normal se utiliza para representar resistencia de materiales, cargas o errores de medición.

En ingeniería eléctrica, la normal es útil en análisis de voltaje, corriente y errores instrumentales.

En energías renovables, la binomial puede modelar eventos discretos de falla y la normal ciertas variaciones continuas alrededor de un valor medio operativo.

13.12 Conclusión

Las distribuciones de probabilidad permiten modelar formalmente variables aleatorias y constituyen una base indispensable para la inferencia estadística, el control de calidad y la confiabilidad en ingeniería.
